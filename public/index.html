HubSpot MCP Server

This is a Machine Communication Protocol (MCP) server for HubSpot integration.

API Endpoints

  • /ping - Health check endpoint
  • /echo - Debug endpoint that echoes back the request body
  • / - Main endpoint for MCP requests

Authentication

All requests must include a HubSpot access token in one of the following ways:

  • Header: X-HubSpot-Access-Token: your-token-here
  • Request body: {"accessToken": "your-token-here"}
  • Request body: {"hubspotAccessToken": "your-token-here"}

Example Request

curl -X POST https://your-vercel-domain.vercel.app/ \
    -H "Content-Type: application/json" \
    -H "X-HubSpot-Access-Token: your-token-here" \
    -d '{"action": "get_contacts"}'
